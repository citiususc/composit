Class Overview

A Service Match Network (SMN) represents a directed composition match graph model. This network consists of two well-differentiated parts: a leveled structure of service operations with multiple layers that correspond with the levels (or layers) of the directed graph (LeveledServices), and the match relations between concepts (MatchGraph). A Service Match Network contains all the information needed to search for optimal compositions.

There is a default implementation called DirectedAcyclicSMN that holds all the information in memory using hash maps.
