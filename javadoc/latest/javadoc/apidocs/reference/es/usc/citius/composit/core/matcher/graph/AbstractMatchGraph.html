public MatchTable<E, T> partialMatch (Set<E> source, Set<E> target)

Find at most one match from source to target. The first element from source that matches an element from target is recorded. The result does not have all matches between source and target. This method can be used to fast check if each element from target has at least one element from source that match it.

Parameters
source source elements (elements to match).
target target elements (elements to be matched).
Returns
  • a Map with keys = matcher elements and values = Match instances.
