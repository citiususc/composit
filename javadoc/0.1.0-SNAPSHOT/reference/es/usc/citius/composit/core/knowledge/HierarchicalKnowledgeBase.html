WSCXMLKnowledgeBase  

Class Overview

Basic implementation of a hierarchical knowledge base. It provides basic methods to test the relation between concepts. This interface is not intended to replace a real knowledge base system. Use this interface to implement basic superclass/subclass inference reasoning with non standard models (for example, WSC'08 taxonomy format).
